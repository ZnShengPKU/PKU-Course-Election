# 模拟选课系统

## 数据准备（Excel → Parquet）
- 需要将教务处导出的 Excel 课程表处理为一张包含以下列的工作表，并保存为 `课表信息汇总.xlsx`（放在项目根目录）：
  - `课程号`、`班号`、`院系`、`课程名`、`参考学分`、`授课教师`、`上课时间`、`修读对象`
  - 可选列（存在则保留）：`学年学期`、`表格类型`、`内部学期`、`课程英文名`、`课程类别`、`周学时`、`总学时`、`起止周`、`备注`
- 时间字段格式示例（用于解析课程表）：
  - `周一1-2`（每周）
  - `周二1-2单`（单周）
  - `周三3-4双`（双周）
  - 支持多时段：`周一1-2，周三3-4`
- 转换规则：
  - 按 `课程号 + 班号` 分组，合并重复记录的 `修读对象`（去重后以 `，` 连接）
  - 其他字段以该组第一条记录为准
  - 输出为 `courses.parquet`，压缩方式为 `snappy`

## 运行转换脚本
- 将 `课表信息汇总.xlsx` 放到项目根目录
- 运行命令：
  - `python3 convert_data.py`
- 转换成功后，会在项目根目录生成 `courses.parquet`，应用启动时将使用该文件以提升加载速度
